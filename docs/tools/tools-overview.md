Tool manuals:

- [`maestroc.md`](maestroc.md): compile `.mstr` source files into one
  packed `.mstro` artifact.
- [`maestrovm.md`](maestrovm.md): compile or load a bundle, load POSIX
  `.so` extensions, validate bindings, and optionally run an entry
  module.
- [`maestroexts.md`](maestroexts.md): inspect the external function
  bindings required by an artifact.
- [`test-runner.md`](test-runner.md): run the repository’s categorized
  source suites and deep integration suites.
